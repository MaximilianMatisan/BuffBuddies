## ðŸ’ª Get started
### Prerequisites
 - Rust **1.85.0** or newer

### Run the project
 Clone this repository and execute:
```bash
cargo run
```
For optimized build:
```bash
cargo run --release
```
After starting the application choose whether to run it as client or server (localhost)

Currently, BuffBuddies requires two running instances:
- one as the server 
- one as the client
